# Project 6 Hack Assembler

### The Driver file for the program is 'main.c', located in the Project6 file.

## How to Run the Program
#### 1. Open your terminal and navigate to Project6 directory.
#### 2. Enter: make
#### 3. Run the program in your terminal using the format: 
./main ../Program_file/Program.asm 

#### Example: 
./main ../add/Add.asm

#### This will create the hack file in that same Program_file directory
